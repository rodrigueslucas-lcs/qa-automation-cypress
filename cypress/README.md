# QA Automation Project - Cypress

Projeto de automação de testes E2E utilizando Cypress.

## 📌 Cenários cobertos

- Login (sucesso e erro)
- Carrinho (adição de produtos)
- Checkout completo
- Validações de checkout

## 🚀 Tecnologias

- Cypress
- JavaScript
- Page Object Model (POM)

## ▶️ Como rodar o projeto

```bash
npm install
npx cypress open
